# 🏦 ArcBank

**Web3 Stablecoin Management & Identity Protocol**

*Built on Arc L1 · USDC / EURC · Decentralized Identity*


---

## Overview

ArcBank is a decentralized, enterprise-grade stablecoin management application built on the **Arc L1 Network**. It simplifies Web3 transactions by combining seamless USDC/EURC payments with a custom on-chain identity system — making blockchain interactions as intuitive as traditional fiat banking.

---

## Features

**Arc Name Service (ANS)**
Developed and deployed a custom smart contract that allows users to register permanent `.arc` domains (e.g., `name.arc`), replacing complex wallet addresses with a personalized Web3 identity.

**Domain-Resolved & Batch Transfers**
The payment engine integrates directly with the ANS contract. Users send funds by typing a `.arc` domain — the system resolves the exact address automatically. A Batch Transfer feature allows sending funds to multiple addresses or domains simultaneously in a single transaction.

**Automated Payment Links**
Users generate shareable payment request URLs. When a sender opens the link, the app automatically parses it and pre-fills the recipient's address, asset (USDC/EURC), and amount.

**Daily GM Protocol**
An on-chain engagement feature where users execute a zero-value smart contract transaction to establish and track a verifiable daily activity streak on the blockchain.

**Arc Pass — Digital Identity Card**
The protocol dynamically generates a holographic, downloadable Web3 ID card for the user — visually verifying their `.arc` domain, wallet address, and on-chain GM streak.

**Real-Time Portfolio & History**
The dashboard fetches live USDC and EURC balances and displays a transparent, verifiable transaction history pulled directly from the Arc blockchain.
